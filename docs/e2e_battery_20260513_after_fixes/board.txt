# Análise: Orquestração vs Modelo Único

**A vantagem decisiva não é o paralelismo, é a ITERAÇÃO:** 5 LLMs permitem feedback loops contínuos onde cada agente refina a saída do anterior, capturando nuances que um modelo isolado (por maior que seja) teria apenas uma tentativa para acertar.

**Segundo: diversidade de perspectivas** - um Opus isolado otimiza para uma distribuição média; 5 agentes especializados em paralelo cobrem outliers, edge cases e domínios específicos que exigiriam prompts cada vez mais complexos no modelo único.
